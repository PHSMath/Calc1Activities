\noindent
This activity will challenge you to apply the Chain Rule in situations where we have incomplete information about the functions involved, or where the functions involved are given as graphs or tables rather than formulas. 

\subsection*{Warmup}
Write down the derivative formulas for $\ln(x)$, $\arcsin(x)$, and $\arctan(x)$. 

\vspace{1in}

\subsection*{Basic practice}

Find the derivative of each of the following. Check your work with Wolfram Alpha. 

\begin{enumerate}
    \item $h(x) = x^2 \ln(x)$
    \item $p(t) = \dfrac{\ln(t)}{e^t + 1}$
    \item $s(y) = \ln(\cos(y) + 2)$
    \item $z(x) = \tan(\ln(x))$
    \item $m(z) = \ln(\ln(z))$
    \item $j(r) = \arctan(e^r + \ln(r))$
\end{enumerate}

\vskip

\subsection*{Advanced practice}

Find the derivatives of each of the following. If we run out of time, work these and check with Wolfram Alpha. 

\begin{enumerate}
    \item $f(x) = x^3 \arctan(x) + e^x \ln (x)$
    \item $p(t) = 2^{t \arcsin(t)$
    \item $h(z) = (\arcsin(5z) + \arctan(4-z))^{27}$
    \item $s(y) = \cot(\arctan(y))$
    \item $m(v) = \ln(\sin^2(v) + 1)$
    \item $g(w) = \arctan \left( \dfrac{\ln(w)}{1+w^2} \right)$
\end{enumerate}
